v1.2 RuntimeUpdated
Table of Contents

    v1.2.0

    • Added runtime module implementations for neural, dsa, ai, data, and image.
    • Upgraded dsa.dijkstra to full weighted shortest-path with path reconstruction.
    • Added practical Sequential neural workflow with fit/evaluate/predict behavior.
    • Updated REPL styling to white prompt with animated spinner.
    • Published release assets for Linux and Windows (deb, exe, installer).
    • Updated docs pages to reflect shipped runtime behavior and examples.

    v2.0.0

    • Added 20 built-in modules including dsa, neural, ai, data, viz, image, web, db, crypto, fs, async, test, math, audio, cloud, game, web3, iot, robot, quantum.
    • Added advanced language features: pattern matching, pipe operator, destructuring, enums, generics, interfaces, decorators, macros, null safety, operator overloading.
    • Improved VS Code extension with richer grammar, snippets, hover, completions, and diagnostics.
