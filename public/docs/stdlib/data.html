data Module

v1.2 RuntimeUpdated
Table of Contents

    Overview

    The data module provides lightweight DataFrame operations suitable for ETL scripts, prototype analytics, and feature engineering pipelines in Vibe.

    Current APIs

    DataFrame Construction▼
    • data.DataFrame(map)
    • data.readCSV(path) returns DataFrame placeholder object
    DataFrame Methods▼
    • df.shape() returns [rows, cols]
    • df.columns() returns column names
    • df.head() returns DataFrame
    • df.select(["colA", "colB"])
    • df.describe()

    Example

    vibe
    import data;

fn main() {
    var df = data.DataFrame({
        "age": [21, 33, 27],
        "score": [90, 75, 88]
    });

    print(df.shape());
    print(df.columns());
    print(df.select(["score"]));
}

    Runtime Notes

    Info
    Current runtime DataFrame implementation focuses on stable object semantics and core selection/shape workflows. It is designed to be extended without breaking API shape.
