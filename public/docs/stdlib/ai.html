ai Module

v1.2 RuntimeUpdated
Table of Contents

    Overview

    The ai module currently includes a practical RandomForest API and core evaluation helpers.

    Current APIs

    Model and Metrics▼
    • ai.RandomForest(nEstimators)
    • model.fit(trainX, trainY), model.predict(testX), model.score(testX, testY)
    • ai.accuracy(yTrue, yPred)
    • ai.classificationReport(yTrue, yPred)

    Example

    vibe
    import ai;

fn main() {
    var model = ai.RandomForest(200);

    var xTrain = [[1.0, 0.0], [0.0, 1.0], [1.0, 1.0]];
    var yTrain = [1, 0, 1];

    model.fit(xTrain, yTrain);

    var pred = model.predict([[1.0, 0.2], [0.1, 0.9]]);
    print(pred);
    print(ai.accuracy([1, 0], pred));
}

    Runtime Notes

    Info
    RandomForest methods are implemented for end-to-end workflow testing and examples. Use this API as a stable entrypoint while additional estimators are added.
