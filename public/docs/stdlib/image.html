image Module

v1.2 RuntimeUpdated
Table of Contents

    Overview

    The image module provides lightweight image object workflows for loading, transform metadata, and export operations directly in Vibe.

    Current APIs

    Image Object Lifecycle▼
    • image.load(path) returns image object
    • img.resize(width, height)
    • img.grayscale()
    • img.save(path)
    • img.show()

    Example

    vibe
    import image;

fn main() {
    var img = image.load("cat.jpg");
    img.resize(224, 224).grayscale().save("out.jpg");
    print(img);
}

    Runtime Notes

    Info
    The current runtime image object is optimized for stable method chaining and script orchestration while native pixel pipeline features are expanded incrementally.
