dsa Module

v1.2 RuntimeUpdated
Table of Contents

    Overview

    The dsa module in v1.2 runtime contains mutable Stack, Queue, and Graph objects plus sorting and shortest-path algorithms.

    Import

    vibe
    import dsa;

fn main() {
    var s = dsa.Stack();
    s.push(42);
    print(s.pop());
}

    Data Structures

    Stack and Queue▼
    • dsa.Stack with push, pop, peek, isEmpty, size, clear
    • dsa.Queue with enqueue, dequeue, front, isEmpty, size, clear
    Graph▼
    • dsa.Graph(directed)
    • addVertex(name), addEdge(from, to, weight), neighbors(vertex)
    • dsa.dijkstra(graph, source, target?)
    vibe
    import dsa;

fn main() {
    var g = dsa.Graph(true);
    g.addVertex("A");
    g.addVertex("B");
    g.addVertex("C");
    g.addVertex("D");

    g.addEdge("A", "B", 1.0);
    g.addEdge("B", "C", 2.0);
    g.addEdge("A", "C", 10.0);
    g.addEdge("C", "D", 1.0);

    print(dsa.dijkstra(g, "A", "D"));
}

    Algorithms

    • dsa.quickSort(list) returns a sorted copy
    • dsa.dijkstra graph shortest path with weighted edges

    Runtime Notes

    Implemented
    dijkstra now runs full priority-queue shortest path and returns distance plus path when target is provided.
    Warning
    Ensure edge weights are numeric; non-numeric weights cause runtime conversion errors.
